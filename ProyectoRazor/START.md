## ? ATAJO RÁPIDO

### Para instalar en 10 minutos:

1. **Clonar**
   ```bash
   git clone https://github.com/rhq-omni777/ProyectoRazor.git
   cd ProyectoRazor
   ```

2. **Abrir en Visual Studio**
   ```
   Archivo > Abrir > ProyectoRazor.sln
   ```

3. **Crear BD** (en Consola del Administrador de Paquetes)
   ```powershell
   Add-Migration InitialCreate
   Update-Database
   ```

4. **Ejecutar**
   ```
   Presiona F5
   ```
